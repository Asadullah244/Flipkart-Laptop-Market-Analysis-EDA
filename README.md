# Flipkart Laptop Market Analysis

## Project Overview

This project focuses on analyzing laptop listings from Flipkart using **Selenium Web Scraping** and **Exploratory Data Analysis (EDA)**. The objective is to understand market trends, pricing patterns, brand presence, and the impact of laptop specifications on pricing.

## Problem Statement

The online laptop market offers numerous options with varying specifications, brands, and price ranges, making it difficult for consumers to identify the best value-for-money products. This project aims to analyze Flipkart laptop listings to uncover pricing trends and determine the factors that influence laptop prices.

## Objectives

* Collect laptop data from Flipkart using Selenium Web Scraping.
* Clean and preprocess the collected data.
* Perform Exploratory Data Analysis (EDA).
* Identify pricing trends and market patterns.
* Analyze the impact of RAM, storage, ratings, and discounts on laptop prices.
* Generate actionable insights and recommendations.

## Data Collected

The following information was extracted from Flipkart laptop listings:

* Laptop Name
* Brand
* Price
* Original Price
* Rating
* RAM
* Storage (ROM)
* Discount
* Processor

## Data Cleaning

* Removed duplicate records
* Handled missing values
* Converted data types
* Removed special characters from price columns
* Standardized RAM and storage values
* Extracted processor information

## Exploratory Data Analysis

The analysis included:

* Price Distribution Analysis
* Price Range Analysis
* Top Brand Analysis
* RAM vs Price Analysis
* Correlation Analysis
* Feature Impact Analysis

## Key Insights

* Most laptops are concentrated in the ₹50,000–₹75,000 price range.
* HP has the highest number of laptop listings on Flipkart.
* RAM is the strongest factor influencing laptop prices.
* 8GB and 16GB RAM are the most common configurations.
* Premium laptops with 32GB+ RAM command significantly higher prices.
* Ratings and discounts have minimal impact on laptop pricing.

## Hypothesis Testing

### Null Hypothesis (H₀)

RAM capacity does not significantly affect laptop prices.

### Alternative Hypothesis (H₁)

RAM capacity significantly affects laptop prices.

### Result

The statistical analysis indicated that RAM has a significant impact on laptop prices.

## Conclusion

The Flipkart laptop market is dominated by mid-range laptops, with RAM emerging as the most influential factor affecting pricing. The findings help consumers better understand market trends and make informed purchasing decisions.
